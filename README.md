Smart Kitchen Monitoring System
📌 Overview

This project is a smart kitchen safety system that uses sensors to detect human presence, gas leakage, smoke, and fire accidents.

⚙️ Components Used
PIR Sensor (Motion Detection)
MQ Gas/Smoke Sensor
Flame Sensor
Buzzer
LED / Light
Arduino


🔧 Working
When a person enters → Light turns ON
When gas/smoke is detected → Buzzer alerts
When flame is detected → Buzzer alerts

🚨 Features
Automatic lighting system
Gas and smoke detection
Fire accident alert system
Real-time buzzer notification

🧠 Technology Used
Arduino Programming
Embedded Systems
Sensors Integration

📌 Conclusion

This system improves kitchen safety by providing real-time alerts using sensors and buzzer.
